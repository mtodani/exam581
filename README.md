# 学生の成績管理システム
Eclipseを使って、5人のチームで開発したJavaの動的Webプロジェクト。  
ユーザの認証機能（ログイン・ログアウト）、メニューページ、ヘッダー・フッター・サイドバーは戸谷真人(mtodani)が担当。
	
## システムの概要

専門学校での学生の得点管理を行うシステム。
得点管理をシステム化することにより、成績処理作業を行っている担当者の負荷を軽減させ、業務効率化を図ることを想定したシステム。

## 使用方法
WebContent/scoremanager/index.jspから実行開始。  
但し、システム利用には別途、H2のDB（ユーザ、学生、点数などDB）を使用する必要あり。
